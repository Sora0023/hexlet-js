# hexlet-js

Мой первый проект на Хекслете

## Dependencies

- [Lodash](https://lodash.com)
- Prettier
- [Eslint](https://eslint.org)


## Makefile

1. make install - установка зависимостей. 
```
    $ npm ci
``` 

2. make lint - проверка кода на соответствие правил.
```
    $ npx eslint .
```

3. make lint-fix - исправление ошибок.
```
    $ npx eslint --fix .
```
